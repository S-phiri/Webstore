# Webstore 🛍️

A minimalist e-commerce web application built using **React** and **Supabase**.

## 🚀 Features

- User registration and login
- Product browsing by category (Men/Women)
- Add/remove items to/from cart
- View total cost of items in cart
- Protected checkout flow with alert confirmation
- Modern, responsive design
- Cart state persists within session

## 🛠️ Tech Stack

- React (with Hooks and Context API)
- Supabase (for authentication and data storage)
- React Router
- Custom CSS for styling

To run the project locally:

npm start
